## Personalization Notes

- For Discogs-directory leads, open with: "I found {{store_name}} through Discogs' Philadelphia record store directory."
- Use `Hi {{store_name}} team` unless a contact name is clearly visible on the current public page or independently confirmed. Do not use names inferred from hidden HTML, stale snippets, or third-party profiles.
- For stores without a polished website, keep the line about Discogs already doing the online-store work.
- For stores with strong personality or events, add one sentence naming the shop's character before the demo link.
- For stores with a live beta storefront, ask for feedback instead of asking whether they want a preview.
